# Pooling Units

Hardware implementations of pooling operations.

## Modules

- `max_pool.sv` - Max Pooling (2x2, 3x3)
- `avg_pool.sv` - Average Pooling
- `global_avg_pool.sv` - Global Average Pooling
- `pooling_unit.sv` - Top-level pooling module

## Features

- Configurable kernel size (2x2, 3x3)
- Configurable stride
- Padding support
