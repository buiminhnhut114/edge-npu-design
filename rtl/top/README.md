# Top-level Modules

Top-level integration and wrapper modules.

## Modules

- `npu_top.sv` - NPU Top-level
- `npu_core.sv` - NPU Core (PE Array + Memory)
- `npu_wrapper.sv` - SoC Integration Wrapper
- `npu_pkg.sv` - Package with Parameters
